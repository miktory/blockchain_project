**HOW TO USE**
=====================
1. Fork this repo
2. git clone https://github.com/your_account/project_name.git
3. cd project_name
4. yarn install
5. yarn chain
6. Open second terminal instance and run
   yarn deploy
7. Open third terminal instance and run
   yarn start

App is available on localhost:3000
